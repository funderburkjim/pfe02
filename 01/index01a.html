Polymer2 First Element Tutorial

  1. In custom-element.html,Define an ES6 class that extends Polymer.ELement.
    The name of the element is 'custom-element'.
    The name of the class is CustomElement.
    customElements.define does the definition.
    The required Javascript is obtained by a link import:
       <link rel="import" href="https://polygit.org/components/polymer/polymer-element.html">
  2. The name of the custom element must
    • start with an ASCII letter
    • contain a dash
  3. In this html file, load the custom element with
      <link rel="import" href="custom-element.html">
  4. The registered element is referenced as a normal html tag:
       <custom-element></custom-element>
