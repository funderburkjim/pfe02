Polymer2 First Element Tutorial

In element definition, use Shadow DOM

  1. enclose the element definition in a <dom-module>
    The 'id' of the dom-module should(?) equal the name of the custom element.
  2. The class registration does not (in this example) require a constructor.
  3. Put html for the element inside a <template>
  4. In this html file, load the custom element with
      <link rel="import" href="dom-element.html">
  5. The registered element is referenced as a normal html tag:
       <dom-element></dom-element>
